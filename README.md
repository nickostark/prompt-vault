# Prompt Templates Library (JSON)

A curated collection of reusable, parameterized prompt templates stored in JSON format for easy integration into apps, workflows, and AI agents.

## Categories

Templates are organized by purpose. Current and future categories may include:

- Writing & editing  
- Coding & debugging  
- Research & analysis  
- Planning & productivity  
- Data & spreadsheets  
- Marketing & sales  
- General utilities

## Features

- Structured JSON format (machine-readable & versionable)
- Variable-based templates for easy customization
- Usage examples included where applicable
- Community contributions welcome

## Usage

Load any template, fill in the defined variables, and send the resulting prompt to your preferred LLM.

## Contributing

Pull requests are welcome. Please follow the template format and include at least one example.
